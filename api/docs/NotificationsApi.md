See [Standard Event Categories](https://knowledgecenter.zuora.com/Central_Platform/Notifications/A_Standard_Events/Standard_Event_Category_Code_for_Notification_Histories_API) for all standard event category codes. - If you specify the `eventTypeName` field, the email template is created based on the corresponding custom event. See [Event triggers](https://www.zuora.com/developer/api-reference/#tag/Event-Triggers) for more information about custom events.  The maximum number of email templates that you can create or update by one call is 1,000.
